# iPhone에서 NZ English Trainer 사용하기

## 가장 간단한 방법: Documents 앱

1. `NZ-English-Trainer.html`을 AirDrop, iCloud Drive, 이메일 첨부, 또는 Files 앱으로 iPhone에 옮깁니다.
2. 공유 메뉴에서 **Documents에 복사** 또는 **Documents로 열기**를 선택합니다.
3. Documents 안에서 `NZ-English-Trainer.html`을 탭합니다.
4. 한국어 카드가 보이면 바로 사용할 수 있습니다. 인터넷과 PC 서버는 필요 없습니다.

Files 앱의 빠른 미리보기는 JavaScript를 실행하지 않을 수 있으므로, 파일을 Files에서 미리보기하지 말고 Documents로 가져와 여세요.

## Safari 홈 화면 앱으로 사용하기

Safari는 로컬 HTML을 홈 화면 웹 앱으로 설치할 수 없으므로 최초 접속용 HTTPS 주소가 필요합니다.

1. `NZ-English-Trainer-iPhone-PWA.zip`의 압축을 풉니다.
2. 안에 있는 `index.html`, `manifest.webmanifest`, `sw.js`, `icon.svg`를 같은 폴더로 GitHub Pages 등 HTTPS 정적 호스팅에 올립니다.
3. iPhone Safari에서 공개된 주소를 엽니다.
4. Safari의 **공유** 버튼을 누릅니다.
5. **홈 화면에 추가**를 선택하고 **추가**를 누릅니다.
6. 홈 화면의 **NZ English** 아이콘으로 실행합니다.

첫 접속이 완료된 뒤에는 앱 파일이 캐시되어 인터넷 연결 없이도 학습할 수 있습니다. 복습 기록은 설치한 브라우저/앱 안에만 저장됩니다.
